![LucasDev](img/LucasDev.png)

Sou desenvolvedor **Full Stack** com foco em **Backend**, trabalhando principalmente com:

- **Node.js**
- **TypeScript**
- **NestJS**

Tenho experiência no desenvolvimento de **APIs REST**, aplicando boas práticas como:

- Arquitetura em camadas (**Controller, Service e Repository**)
- Validação de dados
- Autenticação e autorização com **JWT**
- Integração com bancos de dados relacionais

Minha jornada na programação começou com **Python**, linguagem que fortaleceu minha base em lógica de programação e desenvolvimento de software.

Atualmente, estou focado em construir aplicações mais próximas de cenários reais, buscando evoluir continuamente como desenvolvedor backend.
